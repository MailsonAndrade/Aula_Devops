Comandos Essenciais do Git

  • Inicializar um repositório: git init
  • Adicionar arquivos ao staging: git add .
  • Criar um commit: git commit -m "mensagem do commit"
  • Verificar o status do repositório: git status
  • Ver histórico de commits: git log
  • Definir um repositório remoto: git remote add origin "link-do-repositorio"
  • Enviar código para o GitHub: git push -u origin main

O Git é uma ferramenta essencial para controle de versão, permitindo que você gerencie seu código de forma eficiente e colabore com outras pessoas no desenvolvimento de software.
